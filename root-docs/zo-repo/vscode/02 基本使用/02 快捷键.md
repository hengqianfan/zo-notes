## :keyboard: 基本

### 保存文件

`ctrl` + `S`   

## :keyboard: 光标与跳转
### 多处输入光标

`ctrl` + `alt` + `↑` 或 `↓`

## :keyboard: 窗口相关

### 关闭当前打开的窗口

`ctrl` + `F4`   

`ctrl` + `Fn` + `F4`   

::: warning  <Badge type='danger'>差异提示</Badge>

> 因为各厂商的键盘FN键会内置快捷功能，避免冲突可能会需要同时按下 `Fn` 键

:::

### 关闭所有打开的窗口

`ctrl` + `K`  之后  `ctrl` + `W`

### 退出、关闭编辑器

`ctrl` + `shift` + `W`   

## :keyboard: 视图相关

### 侧边栏的显示与隐藏

`ctrl` + `B`
