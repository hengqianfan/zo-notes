### Node

::: details JS代码运行时

- 官网 [enter](https://nodejs.org/en){target="_blank"}

1. 更换镜像源


```shell
npm config set registry https://registry.npmmirror.com
```
2. **开放脚本权限**

> 使用管理员权限的 `shell` 选择 Y 或者 A

```shell
set-ExecutionPolicy RemoteSigned
```

3. 必装的全局包

```shell
npm i nodemon -g
```

```shell
npm i pnpm -g
```
