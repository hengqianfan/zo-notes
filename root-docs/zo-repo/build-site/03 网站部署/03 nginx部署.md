
## 1. 在 centOS 配置 nginx

## 1.1 下载 nginx

```shell
yum install nginx
```
## 1.2 配置 nginx

- 在 etc 中找到 

```shell
service nginx start
```
```shell
service nginx restart
```

## 2. 在Ubuntu配置Nginx



### 2.1 下载



```shell
# 更新 apt-get 源
sudo apt-get update
# 安装 nginx
sudo apt-get install nginx
# 安装后，将会自动开启 nginx 服务，
# 浏览器输入服务器IP即可查看初始页面
```
