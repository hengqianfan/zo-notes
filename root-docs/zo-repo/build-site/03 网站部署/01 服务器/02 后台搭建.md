### :two:服务器上通过下载连接下载node安装包
::: tip  步骤演示
- **服务器上进入下载目录**
- 在该文件中下载安装包
```
wget https://nodejs.org/dist/v20.11.0/node-v20.11.0-linux-x64.tar.xz
```

```shell
sudo mkdir -p /root/appdata/nodejs
```

```shell
sudo tar -xJvf node-v20.11.0-linux-x64.tar.xz -C /root/appdata/nodejs
```
