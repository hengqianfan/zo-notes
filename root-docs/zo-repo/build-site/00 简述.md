# 建站避坑指引

## 了解需求

::: info  <Badge type='info'>tip</Badge>
**每个人的需求不同，学习能力不同**  

**关于建站，自然应根据具体情形来选择方法**  

**切勿，脑子一热，就去购买域名和服务器**  

**然后发现，弄不来，懒得弄，和没必要弄**  

:::




## 避坑小杂平台


如果你使用的百度引擎，你搜索建站会出来很多建站广告  

这些都是中小公司花钱买的广告  

这些所谓的建站平台不仅广告繁杂，你做出的网站也会限制许多  

如果，你只是搭建静态页面   

虽然把网站放在 `github` 等类似的网页托管平台   

依然是寄人篱下，但二者区别还是挺大的
