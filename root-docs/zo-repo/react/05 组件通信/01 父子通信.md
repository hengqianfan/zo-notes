## 父传子

1. 父组件传递数值 子组件绑定属性
2. 子组件接收数据 props

::: info  <Badge type='info'>props</Badge>

- props 可以传递任意数据

- props 是只读对象，子组件只读取，数据要在父组件中修改
:::

## 插槽

```javascript
function son(props) {
    return (
        <div>this is son,{props.children}</div>
    )
}

function appp() {

    return (
        <div>
            <son>
                <span>插槽</span>
            </son>
        </div>
    )
}
```
