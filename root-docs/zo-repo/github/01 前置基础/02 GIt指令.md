## 用户设置
### 全局

---

#### 查询全局用户名
```bash
git config --global user.name
```
---

#### 修改全局用户名
```bash
git config --global user.name yourName
```

---

#### 查询全局邮箱
```bash
git config --global user.email

```

---

#### 修改全局邮箱
```bash
git config --global user.email yourEmail@example.com

```

#### 查询当前项目邮箱
