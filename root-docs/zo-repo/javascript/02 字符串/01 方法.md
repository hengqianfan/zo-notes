## 拆分字符串 

- `str.split('分割符号')`

将`字符串`按照分割符拆分成`数组`

例如：`'20240801-今日新闻'` -> `['20240801' , '今日新闻']`

```js

let str = '20240801-今日新闻'

let arr = str.split('-')

console.log(arr);

```
::: danger <Badge type='warning'>注意</Badge>
**空格也可以作为分割符号**

```js

let str = '20240801 今日新闻'
// 注意里面有空格，没有空格就按字拆分👇
let arr = str.split(' ')

console.log(arr);

```
:::
