###### 24031901

## 1. 获取与安装

官网 ：https://alist.nn.ci/zh/

👉 软件开源仓库: [enter](https://github.com/alist-org/alist/releases){target="_blank"} <Badge type='info'>Github</Badge>  

选择自己适用的版本  

- 大多数人是它 `alist-windows-amd64.zip`

- 解压后会得到一个 .exe 文件

- 在该文件目录下，打开 CMD 

```shell
alist server
```
- 打开浏览器

- 输入用户名与密码(用户名为admin)
