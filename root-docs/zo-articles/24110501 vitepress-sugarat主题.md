# vitepress-sugarat 主题

官网：https://theme.sugarat.top/  

仓库：https://github.com/ATQQ/sugar-blog

## 1. 创建项目

### 1.1 初始化

```shell
pnpm create @sugarat/theme
```

### 1.2 安装依赖

::: danger <Badge type='warning'>提示</Badge>
- 先进入项目文件夹
:::

```shell
cd my-blog
```
- 安装依赖

```shell
pnpm install
```
