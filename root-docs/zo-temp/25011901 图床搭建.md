## 前言

搭建图床常常有以下几种方式：

- 利用 Github Page 来搭建 <Badge type='info'>最简单</Badge>

- 利用 七牛云 \ cloudfare 

- 利用 阿里云对象存储

- 自建图床服务器

## 利用 cloudfare

### 0. 准备 paypal 账号

### 1. 登录 cloudfare 

https://dash.cloudflare.com/


- 建议通过谷歌账号登录（如果有），可以一键登录。


### 2. 找到 r2 对象存储


- 在网站的侧边菜单中找到 `R2 object stroage`
